Projet noté Contrôle Continu Patrons de conception
Etudiants: ARSENE Sélian, LERICHEUX Guillaume et ANDRIEUX Brice

Jeu:Pixel perfect

Se mettre au niveau du build.xml :
Lancer le programme : ant run

Nettoyer les dossiers générés par ant run : ant clean

Comment jouer : 
Choisir de jouer contre l'ordinateur(1) ou contre un autre joueur(2). 
(1)Mémorisez les formes affichées, puis essayez de les reproduire.
(2)Un joueur dessine des formes à reproduire, puis le second doit les reproduire. Une partie se joue sur plusieurs manches.

Dessiner une forme :
Bouton Ajouter forme: ouvre une fenêtre de sélection de formes. Une fois la forme sélectionnée, cliquez puis glissez pour dessiner la forme.

Supprimer une forme :
Bouton Supprimer forme: cliquez sur la forme que vous souhaitez supprimer.

Déplacer une forme
Bouton Déplacer forme: cliquez sur une forme et glissez pour la déplacer. 
Le clic droit enfoncé permet aussi de déplacer une forme.

Undo/Redo
Bouton <-: Revenir une action en arrière / "Annuler" la dernière action.
Bouton ->: Refaire une action "annulée".
